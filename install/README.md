# Manual Installation

   **Arch Linux:**
   ```bash
   sudo pacman -S sdl3
   ```

   **Ubuntu/Debian:**
   ```bash
   sudo apt install libsdl3-dev
   ```

   **macOS (Homebrew):**
   ```bash
   brew install sdl3
   ```

   **Windows (vcpkg):**
   ```powershell
   vcpkg install sdl3:x64-windows
   ```

   **FreeBSD:**
   ```bash
   pkg install sdl3
   ```
